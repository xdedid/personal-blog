# VitePress 入门指南

VitePress 是一个基于 Vue 和 Vite 构建的强大静态站点生成器。

## 安装

```bash
npm install -D vitepress
```

## 配置

创建 `.vitepress/config.ts`：

```typescript
export default {
  title: '我的站点',
  description: '一个 VitePress 站点'
}
```

## 编写内容

使用 Markdown 配合 Vue 增强功能：

- 自定义组件
- 代码高亮
- 数学公式

## 部署

VitePress 可以部署到：
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages
